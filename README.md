Business Sales Performance Analytics
 Project Overview
This project analyzes retail sales data to identify key business insights such as revenue trends, top-selling products, and high-value regions.

 Tools Used
Tableau
Data Cleaning Techniques
KPI Analysis

 Key Analysis Performed
Revenue trends over time
Top-selling products
High-value regions by revenue

 Key Insights
Revenue showed fluctuations with peak periods indicating seasonal demand
A few products contributed significantly to total sales
Certain regions generated higher revenue compared to others

 Business Recommendations
Focus on high-performing products
Expand in top revenue regions
Improve strategy in low-performing area.
